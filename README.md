# 🏥 BraTech — Telemedicina Inteligente

> Plataforma de telemedicina com triagem por IA que conecta pacientes a médicos em até 12 minutos.

---

## 📋 Descrição

A **BraTech** é uma solução de saúde digital desenvolvida como projeto da disciplina de **Front-End Design Engineering** da FIAP — Global Solution 2026.

O projeto aborda o problema de acesso à saúde no Brasil, onde mais de **160 milhões de pessoas** não têm acesso real a atendimento médico de qualidade. Nossa plataforma utiliza Inteligência Artificial para realizar triagem de sintomas, conectar pacientes ao especialista correto e emitir receitas digitais com QR Code.

---

## 🚀 Funcionalidades

- **Simulador de Triagem por IA** — selecione sintomas e receba diagnóstico de prioridade e especialista indicado
- **Fluxo completo da plataforma** — Cadastro → Triagem IA → Consulta → Receita → Follow-up
- **Formulário de contato** com validação JavaScript completa
- **Menu hambúrguer** responsivo em todas as páginas
- **Contadores animados** na página inicial
- **Modal de demonstração** interativo
- **Design responsivo** — mobile (320px), tablet (768px) e desktop (1366px+)

---

## 🛠️ Tecnologias Utilizadas

| Tecnologia | Uso |
|------------|-----|
| HTML5 semântico | Estrutura das páginas |
| CSS3 (Flexbox + Grid + Media Queries) | Layout e responsividade |
| JavaScript Vanilla | Interatividade, validação e simulador |
| Google Fonts (Poppins) | Tipografia |
| Intersection Observer API | Animações de entrada |

> ⚠️ **Sem frameworks externos** — todo o código foi desenvolvido do zero, sem Bootstrap, jQuery, Axios ou qualquer biblioteca não autorizada.
